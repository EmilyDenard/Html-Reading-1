Questions for Class

What does the br tag do?

It creates a line break in text without making a new paragraph.

What is a "self closing tag"?

Self closing tags are empty tags, they have no content and no closing tag.

What does the strong tag do.

It defines texts as important and also makes the text bold.

Where will you find the DOCTYPE tag? What is it used for?

You will find the tag at the top of the page only once. The tag is used for displaying web pages correctly.

A link is defined with an a tag. What does the href attribute specify?

The links destination.

Explain the concept of a "nested element" to someone with a non-tech background.

Nested elements are elements that are able to have other elements between the open and closing tags.

Why is it a bad idea to skip the end tag (or closing tag)?

Most of the time the elements will display wrong but with some tags it will display correctly.

An image is defined with the img tag. What attributes can be provided to this tag, and what are their functions?

The src attribute specifies the path to the image to be displayed. The required alt attribute specifies an alternate text for the image.
